Turtle Runaway 게임 설명
-----------------------
목표: 빨간 거북이는 파란 거북이를 잡고,
     파란 거북이는 최대한 오래 잡히지 않고 버틴다.
점수: 생존한 1초마다 1점, HUD에 Time / Score / Best 표시.
게임 종료: 빨간 거북이가 파란 거북이의 포획 반경에 들어오면 종료.
재시작: R 키를 눌러 즉시 게임을 다시 시작할 수 있다.
